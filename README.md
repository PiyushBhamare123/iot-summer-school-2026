## Final Project

### IoT-Based Milk Quality and Adulteration Detection System

The final project focuses on developing an educational IoT prototype for monitoring milk-quality parameters using sensors and the ESP32.

The system collects sensor readings, processes the data, detects abnormal conditions using experimentally calibrated ranges, and displays or uploads the results to an IoT dashboard.

**Parameters Monitored:**

* Temperature
* pH
* Conductivity / simulated analog input
* Turbidity

> **Note:** A potentiometer used during simulation only generates variable analog input and does not measure actual milk conductivity. A real EC sensor is required for conductivity measurement.
